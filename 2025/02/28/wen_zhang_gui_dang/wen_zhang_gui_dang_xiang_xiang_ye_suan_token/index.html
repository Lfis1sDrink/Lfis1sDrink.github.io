Markdown Content:
这句话是对于姜文电影《让子弹飞》里面一句对白的模仿，原文是“闹钟响之前说不出来脑袋搬家”“哭？哭也算时间”

最近学习AI的过程中，纠正了自己之前的一个错误的认知，我之前认为大模型会在某种hidden layer里面进行某些思考，因此可以通过prompt让大模型直出答案。现在发现这种观点是错误的，LLM的回复不是思考的结果，而是思考本身。 Deepseek-R1这类推理模型会先进行一段思考，然后再把最后的结果输出。从格式上，思考和最终结果是有明显的格式区隔的，这一点在R1论文中有提到。

因此R1回答问题的宗旨也可以总结成：“要多想，想好了再说”。随着训练轮次的增多，思考变长了，回答的质量也提高了。

思考的过程是必要的，而Deepseek产品上把思考和结果分开也是神来之笔，一下子把用户的预期管理住了，思考的过程，让你不至于干等着，而且有时候还挺有意思，但最终呈现的结果才是final answer.

强行做一个不太恰当的比喻，就好比西式牛排店主厨先烤面包端上来，让你垫垫肚子，同时也把炉子的温度搞热了，正好来做主菜。 不烤这个面包，牛排也烤不对味儿（请不要在意这个比喻从烹饪角度是否合理）。从论李飞飞S1那篇论文中看，有的时候甚至会多烤几炉面包再烤牛排，为的是最终牛排的品质。

回扣一下标题，想当然也算token，但这个token，花的值！
