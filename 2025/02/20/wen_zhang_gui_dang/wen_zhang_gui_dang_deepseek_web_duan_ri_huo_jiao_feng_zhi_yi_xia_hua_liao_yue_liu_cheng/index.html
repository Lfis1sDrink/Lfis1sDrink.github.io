Title: DeepSeek Web端日活较峰值已下滑了约六成

Markdown Content:
DeepSeek APP于2025年1月15日官方宣布上线一个月后，2025年2月15日，DeepSeek App端日活用户为3761万，较峰值4179万（2025年2月6日）下降了约一成。与此同时，ChatGPT App端日活在持续攀升，并于2025年2月15日达到近一个月的峰值6101万，从最近一周的数据来看，DeepSeek与ChatGPT在App端的日活差距在增大。

2025年2月15日，DeepSeek Web端日活用户为923万，较峰值2477万（2025年1月28日）下降了约六成。与此同时，就Web端的访问量和日活指标来看，豆包、Kimi和通义均实现了翻倍增长。

根据非凡产研的独立分析，DeepSeek 官方客户端用户流失的原因主要有以下几点：
- 企业内部部署开源模型：许多企业，尤其是国央企，部署了DeepSeek的开源模型，使得用户无需登录官方客户端即可使用相关服务，导致用户对Web端和App端的依赖度降低。
- 互联网应用接入：许多有巨量用户的互联网应用接入了DeepSeek，如微信、百度搜索、钉钉、飞书等。这些应用的用户可以直接在应用内使用DeepSeek的功能，无需访问DeepSeek Web端，进一步导致了Web端用户的流失。
- 服务稳定性问题：DeepSeek Web端和App端时常出现服务不可用的情况，如服务器繁忙、响应延迟甚至崩溃，严重影响了用户体验，导致用户流失。
- 热度正常下滑：在经历了一段时间的热度攀升后，DeepSeek的用户增长速度开始放缓，这是市场饱和的正常现象。同时，其他平台如豆包、Kimi和通义等也在不断优化和提升自身服务，吸引了更多用户，导致DeepSeek的用户份额被分流。

特别说明：本图表涉及的AI产品流量数据基于多源数据整合及非凡产研智能算法处理，监测范围严格限定于用户通过官方应用及网站的直接访问行为。具体而言，本数据仅涵盖用户通过网站及原生应用（如ChatGPT官方APP及chatgpt.com）直接访问AI产品的行为，不包含以下接入方式产生的流量数据：浏览器插件/扩展程序、桌面客户端软件、微信/支付宝小程序生态、Discord等第三方平台内嵌服务、开源模型本地化部署、API接口调用及其他非直接访问场景，本数据聚焦于AI产品的核心端侧访问行为监测，旨在客观反映主流用户终端的直接使用情况。
