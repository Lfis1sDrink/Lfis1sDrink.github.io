通过 Api 接口调用使用deepseek R1

1:注册硅基流动（siliconflow)账号，获取2000万token免费额度。一般一次对话调用1万以内的token额度，理论上可以对话上千次，个人感觉熬到deepseek官网好用起来应该够用了。😂注册网址：https://cloud.siliconflow.cn/i/NRbfBgVK
2:可以直接在硅基流动网页上使用，但是网页版不能保存对话历史，所以继续
3:下载chatbox，手机上电脑上都有对应的版本。应用商城搜不出来的话，官方网址:https://chatboxai.app/zh
4:根据图1图2几张截图复制硅基流动的API （不要发给别人🤫自己用就行了），图4开始在chatbox上设置api调用模型，记得模型不看选pro开头的。pro是充值用户，我们是免费用户。
5:这两天用得挺流畅的。
